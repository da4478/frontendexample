How to use

Keyboard shortcuts

A – Add to list. Adds new box to list where selected box appears. Cannot add to main box

I – Insert new box

E – Edit selected box

S – Go to menu

M – Move selected text to another box

Delete or D – Delete selected box

Esc – Cancel edit

up/down/left/right arrow – Move mind map

Tab – Moves from one box to next. If box is hidden will make it visible.



Mouse actions

Double click to edit

Single click to select

Click and drag to move box

Click and drag into empty space to move whole map

Select mind map to delete

Enter picture width in PX or leave empty for auto
